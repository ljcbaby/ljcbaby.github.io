从 Git 到 GitHub

发表于 2020-07-20 更新于 2020-07-24 阅读次数：
这儿，程序员的开始！

Git 是什么


Git 是目前世界上最先进的分布式版本控制系统（没有之一），可以跟踪纯文本文件的变化的。
最初的 Git 是为了更好的维护 Linux Kernel 的源代码，但在其出现之后，它迅速成为最流行的分布式版本控制系统。
Git 的常用命令不多，熟悉之后就很容易上手，而且还可以配合各个 GUI 使用。
关于 Git 的用法，可以参考 Git 本身的帮助信息和廖雪峰的 Git 教程，快速入门 So easy.

git

一个免费和开源的分布式版本控制系统

GitHub

最大的开源协作社区

Github

在 Github 出现之前，程序员们交换各自的 Git 仓库并不方便，Github 的出现便很好的解决了这个问题，利用 Git 极其强大的克隆和分支功能，广大程序员真正可以第一次自由参与各种开源项目了。
利用 GitHub，广大程序员可以自由的参与项目，甚至是基于 GitHub 开发项目。配合 jsDelivr，GitHub 还可以成为免费的图床和文件床。
但 Github 并不是唯一的开源协作社区，还有与 GitHub 类似的网站，如 GitLab，国内的 Gitee 等，自己利用 Gitea 搭建自己的 Git 在线平台也是可以的。
