技术信息

发表于 2020-04-08 更新于 2020-08-05
如果需要站点和服务器的一些技术信息，请看这里。

博客

生成器：Hexo
主题：NexT
CI服务：GitHub Actions

文件加载结构：

  • HTML主文件
    • 源站：GitHub Pages
    • 境内：腾讯云CDN (大佬别打，要钱的)
    • 境外：Cloudflare
  • 资源文件、大部分图片: GitHub + jsDelivr

评论

本站的评论使用的是Valine
用的是Gravatar或QQ头像作为评论列表头像。
如果需要修改自己的头像，请自行前往Gravatar或修改自己的QQ头像。
评论的时候，留下在Gravatar登记的邮箱或QQ号即可。
请和谐评论，如需删除评论请用评论的邮箱联络这个邮箱。

服务器

现有两台服务器，坐标分别是上海和Los Angeles
服务器系统：Ubuntu 18.04 LTS
环境：宝塔面板最新测试版
Nginx + PHP最新版
