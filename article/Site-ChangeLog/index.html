站点更新日志

发表于 2021-06-06 更新于 2022-02-22 阅读次数：

记录一下这个站点的变更历史

2022

01.31

临时切换静态资源公共 CDN 到 Zhimg
启用 cdn.ljcbaby.top 到腾讯云 CDN，用于图床和静态文件。

本次 jsDelivr 事故对本站的影响不小，目前的处理仍是临时性的，后续我在考虑将静态文件版本化，同时写一个 sw.js 来提高可靠性并节省流量。

02.22

Zhimg 似乎有了额外限制，再度逃难到 npm.elemecdn.com.

2021

02.11

更换域名，启用阿里云

06.01

启用 Twikoo 评论系统与访问量统计，放弃旧数据

06.06

关闭 Twikoo 的统计功能
页面访问量统计也交给不蒜子

08.20

加入十年之约

2020

疫情网课期间，水过备案，采用 Hexo，重新出发，部署至 Github Pages。

7 月，原始仓库同步至 Github 并开始采用 Travis CI。

后启用腾讯云 +Cloudflare 多点部署

08.04

由于 Valine 评论系统存在安全漏洞，临时关闭评论

详见：发送请求中响应内容明文暴露评论者 IP、邮箱等隐私内容・Issue #336・xCss/Valine

切换至 Github Action

08.08

启用 DisqusJS 为新的评论系统，原有的访问量统计交由 LeanCloud Visitor 负责处理

10.17

感谢云游君的方案，成功把原有评论导入

12.12

准备启用阿里云 CDN

2019

购入域名、服务器，建立了一个 WordPress 站点。

后因备案问题被屏蔽
